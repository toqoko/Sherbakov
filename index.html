Кроссворд СП

Горизонталь

  1. 1. Процессорный ресурс для выполнения команд(ЦПУ)
  2. 2. Блок кода, выполняющий определенную задачу(Функция)
  3. 3. Структура данных для хранения набора элементов(Массив)
  4. 5. Метод, при котором функция вызывает сама себя(Рекурсия)
  5. 8. Текстовый редактор, используемый программистами(Вим)
  6. 11. Технология виртуализации серверов и приложений(Контейнеризация)
  7. 13. Проццес нахождения и исправления ошибок в коде(Отладка)
  8. 14. Система контроля версий(Гит)
  9. 15. Настройка, расширяющая его функциональность(Плагин)
  10. 16. Способ передачи данных через интернет(Протокол)
  11. 18. Совокупность инструкций для выполнения задачи(Алгоритм)
  12. 21. Процесс преобразования исходного кода в машинный код(Компиляция)
  13. 22. Модель данных, состоящая из узлов и связей между ними(Граф)
  14. 23. Программа для управления БД(СУБД)

Вертикаль

  1. 1. Стурктура, выполняющая набор инструкций несколько раз(Цикл)
  2. 2. Структура для отображения содержимого веб-страниц(Фрейм)
  3. 4. Устройство, предоставляющее услуги другим компьютерам в сети(Сервер)
  4. 6. Переменная, содержащая только одно из двух возможных значений(Булево)
  5. 7. Шаблон для создания объектов в ООП(Класс)
  6. 9. Способ хранения и доступа данным с использованием индексов(Индексация)
  7. 10. Проццес проверки программы на наличие уязвимостей(Тестирование)
  8. 12. Служба, позволяющая управлять файлами удаленно через интернет(Облако)
  9. 17. Механизм повторного использования кода, связанный с классами(Наследование)
  10. 19. Программный код выполняемый на стороне клиента(Скрипт)
  11. 20. Программа для защиты компьютера от вирусов(Антивирус)
